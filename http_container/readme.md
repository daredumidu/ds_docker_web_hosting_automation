Step1: create the initial image.

follow instrctions on "https://hub.docker.com/_/httpd" for creating the image. 

Use "dockerfile" to create the base image


Step2: use the initial image to host the webpage. Use the "container_build_run.py" file to create the updated image.

Use "dockerfile" to create the container

For every update, rerun the "container_build_run.py" file.
